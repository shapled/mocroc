# Croc Mobile

基于 Fyne 框架的 Croc 工具移动端 UI 实现。

## 项目特性

- **纯 Go 实现**: 使用 Fyne 跨平台 GUI 框架 + Croc 核心库
- **移动端优化**: 专为触摸设备设计的简洁界面
- **核心功能**: 文件/文件夹发送、文本分享、二维码传输
- **智能过滤**: 自定义忽略规则，避免发送无用文件
- **实时反馈**: 传输进度、状态显示、历史记录

## 开发环境要求

- Go 1.21+
- Fyne v2.4.5+
- Croc v10.0.14+

## 项目结构

```
.
├── main.go                 # 应用入口
├── internal/
│   ├── ui/                # UI 层
│   │   ├── main.go        # 主界面框架
│   │   └── tabs/          # 标签页
│   │       ├── send.go    # 发送界面
│   │       ├── receive.go # 接收界面
│   │       └── history.go # 历史记录
│   └── croc/              # Croc 集成层
│       ├── mobile.go      # 移动端适配
│       ├── client.go      # 客户端封装
│       └── events.go      # 事件处理
├── go.mod
└── README.md
```

## 运行项目

### 开发测试
```bash
# 安装依赖
go mod tidy

# 运行桌面版本（开发测试用）
go run .
```

### 移动端打包
```bash
# Android
fyne package -os android -appID github.shapled.mocroc

# iOS
fyne package -os ios -appID github.shapled.mocroc
```

## 技术说明

### 核心架构
- **UI 层**: Fyne 组件构建用户界面
- **业务层**: 封装 Croc 功能为移动端 API
- **通信层**: 事件驱动的异步通信

### 移动端适配
- 简化的配置选项（隐藏技术细节）
- 触摸友好的交互设计
- 智能的默认配置
- 针对移动端的忽略规则

### 设计原则
- 简洁优先：隐藏复杂的技术细节
- 性能优先：优化传输速度和资源使用
- 容错性好：友好的错误提示和恢复机制
